# 文章簡介

vBulletin是一個強大，靈活並可完全根據自己的需要定制的論壇程序套件。 vBulletin使用目前發展速度最快的Web腳本語言編寫:PHP，並且基於以高效和疾速著稱的數據庫引擎MySQL。 vBulletin是世界上用戶非常廣泛的PHP論壇，很多大型論壇都選擇vBulletin作為自己的論壇應用軟體。
